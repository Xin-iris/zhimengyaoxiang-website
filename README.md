## 打开方式

1. 在 VS Code 中打开本文件夹：
   `/Users/xinyichang/Documents/Codex/2026-05-06/files-mentioned-by-the-user-16119759`
2. 打开 `index.html`
3. 右键选择 `Open with Live Server`，或直接在浏览器中打开 `index.html`

## 已包含内容

- 顾客 DIY 瑶织锦：纹样、配色、尺寸、工期、价格联动
- 平台交易流程：生成创意、可视化编辑、确认支付、织娘交付
- 织娘后台视角：订单排期、织造说明、交易保障
- 单独宣传页：`promotion.html`，突出云织客 logo、品牌口号和传播场景
- 单独预约页：`reserve.html`，支持体验/咨询/礼盒/研学预约和价格估算
- 真实资料图片：保留 PDF 中的纹样图，并补充江华/永州政府公开资料图片
- 宣传视觉：保留高级主视觉和商品宣传图，用于氛围展示，不冒充真实纹样资料
- 主页改版：参考新中式品牌官网的表达方式，用大图、短句、留白和章节入口承载高级感，详细说明放入宣传页

## 主要文件

- `index.html`：网页结构
- `promotion.html`：品牌宣传页面
- `reserve.html`：预约页面
- `styles.css`：高级视觉风格与响应式布局
- `script.js`：DIY 交互逻辑
- `assets/`：网页使用的图片资产
